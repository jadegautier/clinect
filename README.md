# Clinect — ClinicalTrials.gov Search (Team Project)

Data engineering + web app project that extracts clinical trial data from ClinicalTrials.gov, cleans/standardizes fields, and serves searchable results through an API and UI.

## Repo Structure
- `/frontend` — Next.js UI (search, filters, results)
- `/backend` — Flask API + ETL + database integration

## Tech Stack
Python • Flask • SQL • PostgreSQL • MongoDB • Neo4j • Next.js • TypeScript • Docker

## How to Run
1) Start databases (Docker/local)
2) Run ETL to populate DB(s)
3) Start Flask API
4) Start Next.js frontend
